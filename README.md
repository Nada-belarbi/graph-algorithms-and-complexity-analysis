# Graph Algorithms and Complexity Analysis

This repository contains implementations and experimental analysis of classical algorithms studied in algorithm design and graph theory courses.

The goal of the project is to explore algorithm behavior, computational complexity, and performance through practical Python implementations and empirical evaluation.

---

## Repository Structure

The project is organized into two main branches:

### 1. Dijkstra Shortest Path
Branch: `dijkstra-shortest-path`

Implementation and analysis of Dijkstra’s algorithm for computing the shortest path in weighted graphs.

Content:
- Python implementation of graph structures and Dijkstra's algorithm
- Jupyter notebook demonstrating algorithm behavior
- Example graphs and test cases

Concepts covered:
- Graph theory
- Shortest path algorithms
- Priority queues
- Algorithm correctness testing

---

### 2. Sorting Algorithms Complexity Analysis
Branch: `sorting-algorithms-analysis`

Empirical analysis and comparison of classical sorting algorithms.

Implemented algorithms include:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Heap Sort
- Merge Sort
- Quick Sort

Content:
- Python implementations of sorting algorithms
- Benchmark experiments
- Complexity analysis
- Visualization of execution time

Concepts covered:
- Time complexity analysis
- Algorithm benchmarking
- Empirical vs theoretical complexity
- Data size scaling experiments

---

## Technologies Used

- Python
- Jupyter Notebook
- NumPy
- Matplotlib

---

## Purpose

These projects were developed as part of an academic course on algorithm design and complexity analysis.

They demonstrate practical implementations of classical algorithms together with experimental evaluation of their computational behavior.
